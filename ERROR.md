## Το παρακάτω αρχείο αναγράφει όλα τα πιθανά σφάλματα της ΓΛΩΣΣΑΣ

### Σφάλμα Ονόματος προγράμματος - Συντακτικό Σφάλμα
```
ΠΡΟΓΡΑΜΜΑ
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```
```
Σφάλμα στην γρ. 1, αρ <ονομα αρχείου>: Δεν δόθηκε όνομα στο πρόγραμμα.
1. ΠΡΟΓΡΑΜΜΑ
.............^^^^^^
```
___
### Σφάλμα τέλος προγράμματος - Συντακτικό Σφάλμα
```
ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Δ
```
```
Σφάλμα στην γρ. 1, αρ <ονομα αρχείου>: Δεν τελειώνει το πρόγραμμα. Δεν βρέθηκε το ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ.
1. ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Δ
....................^^^^^^
```
___
### Σφάλμα μεταβλητής - Συντακτικό Σφάλμα
```
ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Α
    ΤΕΣΤ <-- 56
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```
```
Σφάλμα στην γρ. 2, αρ <ονομα αρχείου>: Δεν υπάρχει η εντολή "ΑΡΧΗ"
2.     ΤΕΣΤ <-- 56
^^^^^^
Σφάλμα στην γρ. 2, αρ <ονομα αρχείου>: Δεν έχει οριστεί η μεταβλητή "ΤΕΣΤ"
2.     ΤΕΣΤ <-- 56
......^^^^^^
TIP: Όρισε την μεταβλητή στις ΜΕΤΑΒΛΗΤΕΣ.
```
___
### Σφάλμα Αρχής - Συντακτικό Σφάλμα

```
ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Α
ΜΕΤΑΒΛΗΤΕΣ
    ΑΚΕΡΑΙΕΣ: ΤΕΣΤ
    ΤΕΣΤ <-- 56
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```
```
Σφάλμα στην γρ. 4, αρ <ονομα αρχείου>: Δεν υπάρχει η εντολή "ΑΡΧΗ"
4.     ΤΕΣΤ <-- 56
^^^^^^
```
___
### Σφάλμα Τύπου μεταβλητής - Συντακτικό Σφάλμα
```
ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Α
ΜΕΤΑΒΛΗΤΕΣ
    ΑΚΕΡΑΙΕΣ: ΤΕΣΤ
ΑΡΧΗ
    ΤΕΣΤ <-- ΑΛΗΘΗΣ
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```
```
Σφάλμα στην γρ. 5, αρ <ονομα αρχείου>: Η μεταβλητή ΤΕΣΤ είναι τύπου ΑΚΕΡΑΙΑ, δεν είναι τύπου ΛΟΓΙΚΗ
5.     ΤΕΣΤ <-- ΑΛΗΘΗΣ
................^^^^^^
```
___
