## Το παρακάτω αρχείο αναγράφει όλα τα πιθανά σφάλματα της ΓΛΩΣΣΑΣ

```
ΠΡΟΓΡΑΜΜΑ
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```

Σφάλμα στην γρ. 1, αρ <ονομα αρχείου>: Δεν δόθηκε όνομα στο πρόγραμμα.
1. ΠΡΟΓΡΑΜΜΑ
.............^^^^^^
___

```
ΠΡΟΓΡΑΜΜΑ ΘΕΜΑ_Α
    ΤΕΣΤ <-- 56
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ
```
Σφάλμα στην γρ. 2, αρ <ονομα αρχείου>: Δεν υπάρχει η εντολή "ΑΡΧΗ"
2.     ΤΕΣΤ <-- 56
^^^^^^
Σφάλμα στην γρ. 2, αρ <ονομα αρχείου>: Δεν έχει οριστεί η μεταβλητή "ΤΕΣΤ"
2.     ΤΕΣΤ <-- 56
......^^^^^^
TIP: Όρισε την μεταβλητή στις ΜΕΤΑΒΛΗΤΕΣ.
