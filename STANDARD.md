# Σύνταξη της ΓΛΩΣΣΑΣ

## Όνομα προγράμματος
```
ΠΡΟΓΡΑΜΜΑ <όνομα_προγράμματος>
```
Το πρόγρομμα πρέπει να τελειώνει με μια ξεχωριστή γραμμή που λέει `ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ`

## Κώδικας κεντρικού προγράμματος
Ο κώδικας πρέπει να αρχίζει με μία γραμμή που λέει `ΑΡΧΗ` και τελειώνει ο κώδικας εκεί που λέει `ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ`

## Δημιουργία μεταβλητών
```
ΜΕΤΑΒΛΗΤΗ
    <Τύπος μεταβλητών>: <μεταβλητές>
ΑΡΧΗ
```

## Εκχώρηση μεταβλλητών
```
<ονομα_μεταβλητης> <-- <τιμή που είναι ίδιου τύπου με τον τύπο της μεταβλητής>
```

## Συνταξη ΑΝ
- Απλή αν
```
ΑΝ <συνθήκη> ΤΟΤΕ
    ! εντολές
ΤΕΛΟΣ_ΑΝ
```
- Σύνθετη αν
```
ΑΝ <συνθήκη> ΤΟΤΕ
    ! εντολές
ΑΛΛΙΩΣ
    ! εντολές
ΤΕΛΟΣ_ΑΝ
```
- Πολλαπλή αν
```
ΑΝ <συνθήκη_1> ΤΟΤΕ
    ! εντολές
ΑΛΛΙΩΣ_ΑΝ <συνθήκη_2> ΤΟΤΕ
    ! εντολές
ΑΛΛΙΩΣ_ΑΝ <συνθήκη_ν> ΤΟΤΕ
    ! εντολές
ΑΛΛΙΩΣ
    ! εντολές
ΤΕΛΟΣ_ΑΝ
```

## Σύνταξη ΟΣΟ
```
ΟΣΟ <συνθήκη> ΕΠΑΝΑΛΑΒΕ
    ! εντολές
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
```

## Σύνταξη ΜΕΧΡΙΣ_ΟΤΟΥ
```
ΑΡΧΗ_ΕΠΑΝΑΛΗΨΗ
    ! εντολές
ΜΕΧΡΙΣ_ΟΤΟΥ <συνθήκη>
```

## Σύνταξη ΓΙΑ
```
ΓΙΑ i ΑΠΟ 1 ΜΕΧΡΙ Ν ΜΕ_ΒΗΜΑ Β
    ! εντολές
ΤΕΛΟΣ_ΕΠΑΝΑΛΗΨΗΣ
```

## Σύνταξη ΔΙΑΔΙΚΑΣΙΑΣ
```
ΔΙΑΔΙΚΑΣΙΑ <ονομα_διαδικασιας>(ορίσματα)
ΜΕΤΑΒΛΗΤΕΣ
    ! Μεταβλητές όπως στο πρόγραμμα
    ! Τα ορίσματα πρέπει να είναι εδώ
ΑΡΧΗ
    ! Εντολές εκτός από ΓΡΑΨΕ
ΤΕΛΟΣ_ΔΙΑΔΙΚΑΣΙΑ
```

## Σύνταξη ΣΥΝΑΡΤΗΣΗΣ
```
ΣΥΝΑΡΤΗΣΗ <ονομα_διαδικασιας>(ορίσματα):<τύπος συνάρτησης>
ΜΕΤΑΒΛΗΤΕΣ
    ! Μεταβλητές όπως στο πρόγραμμα
    ! Τα ορίσματα πρέπει να είναι εδώ
ΑΡΧΗ
    ! Εντολές εκτός από ΓΡΑΨΕ
    <ονομα_διαδικασιας> ! Για επιστροφή
ΤΕΛΟΣ_ΣΥΝΑΡΤΗΣΗΣ
```

## Εκφράσεις
1. Μαθηματικοί τελεστές
    - +, -, *, /, ^, MOD, DIV
2. Συγκριτική τελεστές
    - \>, <, <=, >=, <> 
3. Λογική τελεστές
    - ΟΧΙ, ΚΑΙ, Η

### Σειρά εκτέλέσης τελεστών εκφράσεων
1. Αριθμός, μεταβλητές
2. Παρενθέσεις ()
3. Συναρτήσεις
4. -, +
5. *, /, MOD, DIV
6. ^
7. \>, <, <=, >=, <> 
8. ΟΧΙ
9. ΚΑΙ
10. Η | 'Η
